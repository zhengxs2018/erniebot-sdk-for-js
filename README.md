# ERNIE Bot SDK for JavaScript

> 非官方 JS-SDK，请勿在生产中使用

可以调用文心大模型的能力，包含文本创作、通用对话、语义向量、AI作图等。

## 参考

- [openai-node](https://github.com/openai/openai-node)
- [ERNIE-Bot-SDK](https://github.com/PaddlePaddle/ERNIE-Bot-SDK)
