# @zhengxs/erniebot-cli

> 非官方 JS-SDK，请勿在生产中使用

可以调用文心大模型的能力，包含文本创作、通用对话、语义向量、AI作图等。

## 安装

```sh
# With NPM
$ npm i -g @zhengxs/erniebot-cli

# With Yarn
$ yarn global add @zhengxs/erniebot-cli

# With PNPM
$ pnpm add -g @zhengxs/erniebot-cli
```

## 使用

```sh
# 开始聊天
$ erniebot

# or
$ erniebot --token xxx
```
